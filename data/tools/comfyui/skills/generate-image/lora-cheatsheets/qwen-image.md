# Qwen-Image LoRA 치트시트

## 프롬프트 운영 원칙

- 문장형 지시 우선
- 태그는 보조
- 관계, 동작, 손 위치, 카메라 구도 같은 문장 정보를 먼저 적는다

## 기록 규칙

- `prompt style` 필드에
  - natural language first
  - mixed
  - tags as support
  같은 식으로 남긴다.
- 부작용 메모에는
  - 과도한 태그 나열 시 성능 저하
  - 장문 지시에서 특정 객체 누락
  같은 해석 문제를 기록한다.
