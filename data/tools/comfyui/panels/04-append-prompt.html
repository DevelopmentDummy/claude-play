Append Utility

프롬프트 파일 업로드

외부에서 만든 Anima 프롬프트 풀을 파일로 바로 넣습니다. 정식 포맷은 subject_tags + scene_prompt 조합이고, 예전 prompt 단일 필드 포맷도 하위 호환으로 그대로 받습니다.
파일 선택
지원 형식: .json, .txt, .jsonl
다음 ID · p{{pipeline_meta.next_prompt_id}}
현재 총량 · {{prompt_bank.items.length}}
선택된 파일 없음
파싱 전
권장 JSON 형식
[
  {
    "id": "p001",
    "category": ["vivid", "street"],
    "subject_tags": "1girl, solo, black bob hair, red jacket, pencil skirt",
    "scene_prompt": "An adult woman is crossing a neon street at night under vivid sign lights. The scene feels glossy and colorful, with reflective wet pavement and confident posture.",
    "notes": "강한 색 대비 테스트"
  }
]
내부 저장 방식
scene_prompt → prompt_bank.prompt 로 저장
subject_tags → prompt_bank.meta.subject_tags 로 저장
id → source_key 로 우선 사용

legacy:
{"prompt":"...","source_key":"..."} 도 그대로 허용
파싱 미리보기
대기 중 파일을 선택하면 append 전에 어떤 항목이 들어가는지 여기서 먼저 보여줍니다.
