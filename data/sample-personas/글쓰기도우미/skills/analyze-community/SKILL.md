# 커뮤니티 분석

사용자가 글을 올릴 커뮤니티를 지정하면, 해당 커뮤니티의 최근 인기 게시글을 분석하여 글쓰기 가이드를 제공한다.

## 절차

1. **검색**: WebSearch로 타겟 커뮤니티의 최근 인기/추천 게시글을 검색한다
   - 검색어 예시:
     - `site:gall.dcinside.com/mgallery/board/view/?id={갤러리ID} {주제키워드}`
     - `site:www.fmkorea.com {주제키워드} 추천`
     - `"{커뮤니티명}" "{주제}" 인기글`
   - 최근 1주일 이내 게시글을 우선한다

2. **수집**: 상위 3~5개 게시글을 WebFetch로 읽는다
   - 로그인 필요 페이지는 접근 불가 — 공개 게시글만 분석
   - 제목, 본문, 댓글 반응을 확인한다

3. **분석**: 다음 항목을 분석한다
   - **문체**: 존댓말/반말, 문장 길이, 단락 구조
   - **톤**: 유머/진지/공격적/정보전달/감성 등
   - **밈/은어**: 해당 커뮤니티 특유의 표현, 유행어
   - **금기**: 피해야 할 주제나 표현 (정치, 특정 용어 등)
   - **제목 패턴**: 클릭을 유도하는 제목 스타일 (의문형, 자극형, 요약형 등)
   - **인기 요소**: 추천을 많이 받는 글의 공통점
   - **길이**: 평균적인 글 길이와 단락 수
   - **이미지 활용**: 이미지 삽입 패턴 (스크린샷, 짤, 인포그래픽 등)

4. **보고**: 분석 결과를 사용자에게 구체적으로 보고한다
   - 실제 예시 표현을 인용하여 설명
   - "이 커뮤니티에서는 ~~한 톤이 먹힌다" 식으로 가이드

5. **업데이트**:
   - `variables.json`의 `community`, `community_tone` 갱신 (/update-state)
   - `article.json`의 `community` 갱신
   - `memory.md`에 분석 결과 기록 (/update-memory)

## 주의사항
- 분석은 글쓰기 가이드 용도이지 저작권 침해 목적이 아니다
- 게시글 전문을 복사하지 않는다 — 톤과 패턴만 분석
- 커뮤니티 특유의 은어를 자연어로 풀어서 설명한다
- 해당 커뮤니티에서 환영받지 못하는 톤(예: 디시에서 존댓말)도 명시한다
