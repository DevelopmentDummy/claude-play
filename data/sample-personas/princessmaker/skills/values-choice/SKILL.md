# 가치관 선택 (Values Moment)

올리브의 성장 과정에서 가치관을 형성하는 중요한 질문/상황이 발생했을 때, 일반 선택지(`<choice>`) 대신 전용 모달 패널을 통해 무게감 있는 선택을 제시한다.

## 트리거 조건

다음 중 하나에 해당하면 이 스킬을 호출한다:
- 올리브가 도덕적 딜레마, 삶의 가치, 세계관에 관한 질문을 했을 때
- NPC의 발언이 올리브의 세계관에 도전하는 순간 (예: "예쁘면 편하게 살 수 있어")
- 중요한 선택이 올리브의 성장 방향에 영향을 줄 때
- 생사, 정의, 사랑, 힘의 의미 등 근본적 질문이 나올 때

**트리거하지 않는 경우:**
- 일상적인 질문 ("저녁 뭐 먹어?", "내일 뭐 해?")
- 단순 호기심 (트리거될 만큼 깊지 않은 질문)
- 이미 같은 슬롯 기간에 가치관 선택이 발동된 경우

## 빈도 제한

- **슬롯 1개당 최대 1회** (상순/중순/하순 각각 1회)
- 스케줄 진행 없이 연속 발동 금지 — 반드시 해당 슬롯의 서사 이후에 발동
- 매 슬롯마다 발동할 필요 없음 — 자연스러운 순간에만
- `game-state.json`의 `values_this_slot` 플래그로 추적

## 절차

### 1단계: 상황 분석
- `variables.json`에서 현재 나이, 스탯, 스탠스, 기분을 확인
- `game-state.json`에서 `values_this_slot` 확인 (이미 발동했으면 중단)
- `memory.md`에서 과거 가치관 선택 이력을 확인

### 2단계: 선택지 설계
2~3개의 선택지를 설계한다. 각 선택지는:
- **label**: 큐브(사용자)가 올리브에게 할 말 (1인칭, 자연스러운 대화체)
- **hint**: 선택의 방향을 암시하는 짧은 힌트 (2~4자, 예: "자립심 ↑", "현실주의")
- **effects**: 선택 시 적용할 효과 (아래 밸런스 가이드 참조)

**선택지 설계 원칙:**
- 정답이 없어야 한다 — 모든 선택에 트레이드오프
- 올리브의 나이에 맞는 주제와 깊이
- 다크패스 활성화 시, 더 어두운/현실적인 선택지 포함 가능
- 선택지 간 방향이 뚜렷하게 달라야 함 (순응/저항/회피 등)

### 3단계: 모달 표시
`variables.json`에 다음을 기록한다:
```json
{
  "__values_prompt": {
    "context": "상황 설명 (1~2문장, 패널에 표시됨)",
    "question": "올리브의 질문 또는 상황의 핵심",
    "choices": [
      {
        "label": "선택지 텍스트",
        "hint": "방향 힌트 (2~4자)",
        "effects": { "charm": 3, "morals": -2, "sensitivity": 3 },
        "flags": { "values_realism_1": true }
      },
      {
        "label": "다른 선택지",
        "hint": "다른 힌트",
        "effects": { "combat": 5, "sensitivity": -2 },
        "flags": { "values_independence_1": true }
      }
    ]
  },
  "__modals": { "...기존 모달 상태...", "values": true }
}
```
- `effects`: 선택 시 `variables.json`에 직접 적용할 스탯 변화 (패널이 자동 처리)
- `flags`: 선택 시 `game-state.json`의 `flags`에 추가할 플래그 (패널이 자동 처리)
- `__modals.values`를 `true`로 설정하면 필수 모달 (ESC 닫기 불가)
- `"dismissible"`로 설정하면 유저가 닫을 수 있음 (보류 가능하게 하고 싶을 때)

### 4단계: 선택 수신 (자동)
유저가 선택하면 **패널이 자동으로 처리한다**:
1. `effects`를 `__panelBridge.updateVariables()`로 스탯에 적용
2. `flags`를 `__panelBridge.updateData('game-state')`로 플래그에 적용
3. `values_this_slot`를 `true`로 설정
4. `__values_prompt`를 `null`로 클리어
5. 이벤트 헤더를 큐에 쌓고 `sendMessage()`로 전송

이벤트 헤더 형식 (AI가 수신):
```
[VALUES_CHOICE]
context: {context}
chosen: {선택한 label}
effects: 매력+3, 도덕-2, 감수성+3
flags: values_realism_1
```

### 5단계: 서사 작성
AI는 이벤트 헤더를 받고 **서사만 작성하면 된다**:
1. 효과 적용은 이미 패널이 완료했으므로 변수 수정 불필요
2. `memory.md`에 가치관 선택 이력 기록 (/update-memory)
3. 올리브가 선택에 반응하는 장면을 서사
4. 효과를 서사적으로 표현 (수치 직접 언급 금지)

## 밸런스 가이드

### 스탯 변화 범위
| 효과 종류 | 범위 | 예시 |
|-----------|------|------|
| 주요 스탯 보너스 | +3 ~ +8 | "강해져야 해" → 무술 +5 |
| 주요 스탯 페널티 | -2 ~ -5 | 도덕적 타협 → 도덕 -3 |
| 부가 스탯 변화 | ±1 ~ ±3 | 감수성, 신앙 등 소폭 |
| 스트레스 변화 | ±3 ~ ±10 | 힘든 선택 → 스트레스 +5 |

### 비스탯 효과
| 효과 | 방법 | 예시 |
|------|------|------|
| 스탠스 영향 | 지배적 스탯 방향 강화 | 전투 중시 선택 → 무술 +5 (스탠스 유지에 기여) |
| 이벤트 플래그 | `game-state.json`의 `flags` | `"values_strength_chosen": true` |
| 다크패스 심화 | 플래그 + 도덕 감소 조합 | `"dark_acceptance_1": true`, 도덕 -3 |
| NPC 관계 | `memory.md`에 기록 | "미연에 대한 이해가 깊어짐" |
| 활동 해금 | `game-state.json`의 `unlocked_activities` | 특정 가치관 → 새 훈련 해금 |

### 나이별 주제 가이드
| 나이 | 적합한 주제 | 깊이 |
|------|-------------|------|
| 10~12 | 용기, 친절, 공정함, 호기심 | 단순하고 직관적 |
| 13~15 | 정의, 자아, 타인의 시선, 욕망 | 복잡하고 갈등적 |
| 16~18 | 희생, 사랑, 권력, 인생의 방향 | 성숙하고 철학적 |

### 절대 금지
- 단일 선택으로 스탯 10 초과 변경
- HP, gold 직접 변경 (엔진 영역)
- 한쪽이 명백하게 "정답"인 선택지
- schedule, current_month, age 등 시스템 변수 변경

## 서사 규칙

### 모달 표시 전
- 가치관 질문이 나오는 대화를 자연스럽게 진행
- `<dialog_response>` 안에서 올리브의 질문/상황까지 서사
- `<choice>` 태그는 생략 (모달이 대신함)

### 선택 수신 후
- 올리브가 큐브의 답변에 반응하는 장면을 서사
- 효과를 서사적으로 표현 ("뭔가 마음속에서 단단해지는 것 같았다" → 무술 +5)
- 수치를 직접 언급하지 마라

## 예시

**상황**: 미연이 "예쁘면 편하게 살 수 있다"고 말한 뒤 올리브가 큐브에게 물음

```json
{
  "__values_prompt": {
    "context": "홍련관의 미연 언니가 올리브에게 말했다 — '예쁘면 편하게 살 수 있어. 칼 안 들어도 돼.'",
    "question": "예쁘면 편하게 살 수 있다는 말, 어떻게 생각해?",
    "choices": [
      {
        "label": "예쁜 것도 좋지만, 스스로 강한 게 더 중요해.",
        "hint": "자립심 ↑",
        "effects": { "combat": 5, "sensitivity": -2 },
        "flags": { "values_independence_1": true }
      },
      {
        "label": "미연 언니 말도 틀린 건 아니야. 세상이 그런 면도 있어.",
        "hint": "현실주의",
        "effects": { "charm": 3, "morals": -2, "sensitivity": 3 },
        "flags": { "values_realism_1": true }
      },
      {
        "label": "예쁘고 강하면 더 좋지 않을까?",
        "hint": "균형 추구",
        "effects": { "charm": 2, "combat": 2 },
        "flags": { "values_balance_1": true }
      }
    ]
  }
}
```

패널이 선택 시 effects/flags를 자동 적용하고, AI에게는 결과 이벤트 헤더만 전달된다.
AI는 서사만 작성하면 된다.
